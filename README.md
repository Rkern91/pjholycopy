# Desenvolvimento na disciplina de TÃ³picos Especiais em Desenvolvimento de Software III a pedido da comunidade.

Projeto composto de por /CLIENT e /SERVER
Para rodar o /CLIENT
1. Acessar a pasta '/client'
2. Rodar 'npm run dev'

Para rodar o /SERVER
1. Acessar a pasta '/server/src'
2. Criar uma copia do ".env_example" para ".env" configurar o dados para acesso ao banco
3. Rodar 'npm run dev'
